QR Code Digital Menu System

A complete web-based restaurant menu platform built with PHP, MySQL, HTML5, CSS3 & JavaScript

13 Menu Items 5 Categories Full Admin Panel QR Generation
Core PHP MySQL HTML5 CSS3 JavaScript Bootstrap 5 XAMPP

System Modules

Authentication

Secure admin login with bcrypt password hashing, session management, and role-based access (Admin/Manager).

Dashboard

Statistics cards showing total items, categories, views, QR codes. Most viewed item and recent updates.

Category Management

Add, edit, delete categories. Set category images, icons, sort order, and toggle active status.

Menu Management

Full CRUD for menu items. Set prices, descriptions, ingredients, allergens, dietary flags, preparation time, calories.

QR Code Generator

Generate unique QR codes for restaurant, table numbers, and categories. Download and print support.

Analytics

Track menu views, most viewed items, daily visitor charts, category breakdown, and peak access times.

Customer Experience

Mobile-First Design

Fully responsive interface optimized for smartphones. Customers scan QR codes and instantly view the menu.

Real-Time Search

Instant search with debounced input. Find any menu item by name or description in milliseconds.

Dark/Light Mode

Customers can toggle between dark and light themes. Preference is saved in local storage.

Category Filters

Filter menu items by category with smooth animated transitions between selections.

Allergen Information

Clear allergen warnings and ingredient lists on every item detail page.

Installation Guide

1
Install XAMPP

Download and install XAMPP from apachefriends.org. Start Apache and MySQL services.

2
Import Database

Open phpMyAdmin at localhost/phpmyadmin, create database "restaurant_menu", and import the database.sql file.

3
Deploy Files

Copy all project files to C:\xampp\htdocs\restaurant-menu\ (Windows) or /opt/lampp/htdocs/restaurant-menu/ (Linux).

4
Access System

Open browser and go to localhost/restaurant-menu/ for the menu, or localhost/restaurant-menu/admin/login.php for admin.
